# PENA-REST-ENGIN CRUD EXAMPLE
Example from pena-rest-engin

## Getting Started
### Environtment Setup
```
mv env.example .env
```

### Installing
#### Development Mode
```
pip install -e .
```

#### Production Mode
```
pip install .
```

### Run Migration
```
pena migrate up
```
or down migration

```
pena migrate down
```
### Running
```
pena http serve
```
