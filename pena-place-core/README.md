# PENA FRAMEWORK
Microservice specialist framework

## Getting Started
### Environtment Setup
```
mv env.example .env
```

### Installing
#### Contribution Mode
```
pip install -e .
```

#### Programmer Mode
```
pip install .
```

### Running
```
place_core http serve
```
